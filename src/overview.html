This document is the API specification for the TLA platform layer RUMBA.
Documentation for controls provided by Rumba Applications can be found here :
- MMP
- Audio
- VoiceMemo(based on new devices abstractions).

RUMBA is also known as "EBL" : Entertainment Business Logic.
The purpose of this layer is to provide high level interfaces to ease the manipulation of multimedia devices.
As those components should be reusable for several customization projects, those interfaces should be flexible enough to support very different clients needs.
